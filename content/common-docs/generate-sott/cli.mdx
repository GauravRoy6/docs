With [Loginradius CLI](https://github.com/LoginRadius/lr-cli/), You can generate SOTT using the simple command below

   ```sh
        $ lr generate-sott -f <FromDate(mm/dd/yyyy)> -t <ToDate(mm/dd/yyyy)> 
		Select a technology
		.....
		.....
		
		sott generated successfully
		AunthenticityToken: <token>
		Comment: <comment>
		Sott: <sott>
		Technology: <tech>
   ```
