With **<a href="https://github.com/LoginRadius/lr-cli/" target="_blank">LoginRadius CLI</a>**, You can get your API Credentials using the simple command below

```sh
$ lr get config
API Key: ###Your API Key###
API Secret: ###Your API Secret###
```
